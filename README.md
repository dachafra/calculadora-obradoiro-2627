# Calculadora de abonos Obradoiro 2026-2027

Calculadora estática para estimar el precio del abono del Obradoiro a partir de zona, tarifa, descuentos y aportaciones de la campaña 26/27.

## Uso local

Abre `index.html` directamente en el navegador. No necesita servidor, instalación ni compilación.

## Despliegue en GitHub Pages

1. Sube este repositorio a GitHub.
2. Entra en `Settings` > `Pages`.
3. En `Build and deployment`, elige `Deploy from a branch`.
4. Selecciona la rama principal y la carpeta `/root`.
5. Guarda los cambios.

GitHub Pages servirá `index.html` como página principal.
